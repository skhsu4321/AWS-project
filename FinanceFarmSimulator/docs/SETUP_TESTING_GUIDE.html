üìã Detailed Setup Instructions

For those who want to understand each step or need manual installation:

1. Install Node.js

Node.js is required to run the development server.

  1. Go to https://nodejs.org/
  2. Download the LTS version (left button, recommended)
  3. Run the installer and follow the setup wizard
  4. Restart your computer after installation
# Verify installation by opening Command Prompt and typing: node --version npm --version

2. Navigate to Project Folder

Open Command Prompt and navigate to your project:

# Replace with your actual path cd C:\Users\YourName\Documents\FinanceFarmSimulator

üí° Tip: You can also Shift+Right-click in the folder and select "Open PowerShell window here"

3. Install Dependencies

Install all required packages (this may take 2-5 minutes):

npm install --legacy-peer-deps

‚ö†Ô∏è Important: The --legacy-peer-deps flag is required to resolve version conflicts between React Native and Expo Router.

4. Start Development Server

Start the app in web mode:

npm run web

The development server will start and automatically open your browser to http://localhost:19006

5. Verify Installation

You should see:

  • The Finance Farm Simulator login screen
  • No error messages in the browser console
  • Smooth animations and interactions

üîß Troubleshooting

Common issues and solutions:

‚ùå "Node.js is not recognized"

Problem: Command prompt says Node.js is not found

Solution:

  1. Download Node.js from nodejs.org
  2. Install the LTS version
  3. Restart your computer completely
  4. Open a new Command Prompt window

‚ùå "npm install" fails

Problem: Dependency installation errors

Solution:

# Clear npm cache npm cache clean --force # Delete node_modules and package-lock.json rmdir /s node_modules del package-lock.json # Reinstall with legacy peer deps npm install --legacy-peer-deps

‚ùå Quick-start.bat window disappears

Problem: The batch file window closes immediately

Solution:

  1. This is normal! The script opens files automatically
  2. Check if browser windows opened with setup guides
  3. If nothing opened, right-click the .bat file ‚Üí "Run as administrator"
  4. Or open Command Prompt manually and run the commands

‚ùå "Port 19006 is already in use"

Problem: Development server can't start

Solution:

# Kill existing processes taskkill /f /im node.exe # Or use a different port npx expo start --web --port 19007

‚ùå App shows blank screen

Problem: Browser shows empty page

Solution:

  1. Open browser developer tools (F12)
  2. Check Console tab for error messages
  3. Try refreshing the page (Ctrl+F5)
  4. Clear browser cache and cookies
  5. Try a different browser (Chrome recommended)
